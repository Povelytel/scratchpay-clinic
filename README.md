# scratchpay-clinic
Coding challenge - my solution for clinic search

## Installation

```bash
$ npm i
```

## Env 

In config can make setting for nodeJs app

## Running the app

```bash
# for start local
$ npm run start:local
```

## cURL for send request

```bash
# Example for search
curl --request GET \
  --url 'http://localhost:4000/api/v1/search/?search=Cli'

```
